Web DHKP Desa Bululawang

About

Web DHKP Desa Bululawang berfungsi untuk mempermudah pendataan dan pencarian data tanah. Fitur-fitur yang ada yaitu tambah data, ubah data, hapus data atau hapus semua data. Selain itu ada fitur eksport dan import data dari file excel dengan format kolom disesuaikan dengan tampilan tabel pada halaman home. Web DHKP juga mempunyai keamanan dengan adanya login menggunakan username dan password
