Web Kasir Toko Bangunan

About

Web Kasir Toko Bangunan berfungsi untuk mempermudah penjualan barang. Terdapat 2 user yaitu admin dan kasir. Admin berfungsi untuk mengelola barang dan melihar riwayat penjualan setiap harinya, riwayat penjualan bisa dieksport menjadi foemat excel. Kasir berfungsi urntuk proses penjualan terdapat fitur search dan scanner barcode, jika semua barang sudah diinputkan tinggal melakukan pembayaran, nantinya kembalian pembayaran akan otomatis ditampilkan. Web Kasir juga mempunyai keamanan dengan adanya login menggunakan username dan password
