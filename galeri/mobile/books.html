Daftar Buku Islami

About

Daftar Buku Islami berfungsi untuk menyajikan daftar buku-buku islami, yang nantinya akan disediakan link untuk bisa langsung membaca secara online ataupun mendownload buku tersebut. Pada Aplikasi Mobile ini terdapat dua user yaitu admin dan pengunjung. Admin sebagai penyedia daftar buku-buku islami harus Login terlebih dahulu, sedangkan pengguna langsung Get Start karena pengunjung hanya dapat melihat dan membaca buku.
